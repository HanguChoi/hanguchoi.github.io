📄 How HTML Files Work

Understand the structure — then move your code from CodePen to GitHub!

1 The Three Parts of Every Webpage

Every webpage is built from three ingredients. Think of them like a house: HTML is the walls & rooms, CSS is the paint & furniture, and JavaScript is the electricity & doors that open.

HTML — Structure CSS — Style JavaScript — Behavior
2 The Anatomy of an HTML File

A complete HTML file has a specific structure. Every section has a job:

<!-- The file always starts here -->
<!doctype html>
<html>
 
  <!-- HEAD: browser settings, title, and CSS -->
  <head>
    <title>My Page Title</title>
    <meta charset="utf-8">
    <!-- CSS goes here, inside <style> -->
    <style>
      body { background: blue; }
    </style>
  </head>
 
  <!-- BODY: everything the user sees -->
  <body>
    <h1>Hello World!</h1>
    <button>Click me</button>
    <!-- JS goes here, at the end of <body> -->
    <script>
      alert("Hello!");
    </script>
  </body>
</html>
head — settings & CSS body — visible content script — JavaScript
Key rule: CSS lives in <style> inside <head>. JavaScript lives in <script> at the end of <body>. Your HTML content goes in between, inside <body>.
3 CodePen Has 3 Panels — GitHub Needs 1 File

CodePen is great for experimenting because it splits HTML, CSS, and JS into 3 separate boxes. But to publish on GitHub Pages, you need to combine everything into one HTML file. Here's how they match up:

CodePen — HTML Panel
<h1>Hello!</h1>
<button>Click</button>
<p>My page</p>
CodePen — CSS Panel
body {
  background: blue;
}
h1 { color: white; }
CodePen — JS Panel
let count = 0;

function click() {
  count++;
}
→
index.html (GitHub)
<!doctype html>
<html>
<head>
 <style>
  ← CSS panel goes here
 </style>
</head>
<body>
 ← HTML panel goes here
 <script>
  ← JS panel goes here
 </script>
</body>
</html>
4 Step-by-Step: CodePen → GitHub
1
In your GitHub repo, create a new file called index.html (or open the existing one you want to update).
2
Copy the template below and paste it into your new file. This gives you the correct skeleton.
3
Go to CodePen. In the CSS panel, select all the code and copy it. Then paste it between <style> and </style> in your file.
4
In the HTML panel, copy all the code. Paste it between <body> and <script>.
5
In the JS panel, copy all the code. Paste it between <script> and </script>.
6
Save the file and commit! Your page should now work on yourusername.github.io — same as CodePen.
5 The Template — Copy This!

Use this as your starting point every time you move from CodePen to GitHub:

index.html — starter template Select all → Copy → Paste into your GitHub file
<!-- Step 1: Always start every HTML file with these two lines -->
<!doctype html>
<html lang="en">

<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Awesome Page</title>

  <!-- Step 2: Paste your CSS panel code here -->
  <style>

    /* your CSS goes here */

  </style>
</head>

<body>

  <!-- Step 3: Paste your HTML panel code here -->


  <!-- Step 4: Paste your JS panel code here -->
  <script>

    // your JavaScript goes here

  </script>

</body>
</html>
Tip: The order matters! Always put <style> in the head, and <script> at the end of the body — this makes your page load correctly and your JavaScript can find all the HTML elements it needs.
