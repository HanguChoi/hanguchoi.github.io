🐙 Your Free Website on GitHub

Create a public website at yourusername.github.io — totally free!

1

Go to GitHub and sign in

Open github.com in your browser. If you don't have an account, click Sign up — it's free.

2

Create a new repository

Click the + button at the top right → New repository.

3

Name it exactly right — this is the magic part

In the Repository name box, type:

yourusername.github.io
Replace yourusername with your actual GitHub username — must match exactly!
4

Set it to Public

Make sure Public is selected (not Private). GitHub Pages only works on public repos for free accounts.

5

Click "Create repository"

Scroll down and click the green Create repository button. Your repo is ready!

6

Add your first file — index.html

Click Add file → Create new file. In the filename box, type index.html.

Then paste this Hello World code in the big text area below:

<!doctype html>
<html>
<body>
  <h1>Hello World! 🎉</h1>
  <p>My website is live!</p>
</body>
</html>
7

Save it — click "Commit changes"

Scroll down, leave the message as-is, and click the green Commit changes button.

8

Wait about 1 minute…

GitHub needs a little time to build your site. Then visit your URL!

🎉 Your site is live!

Open your browser and go to:

https://yourusername.github.io

Every time you edit index.html and commit, your site updates in about a minute.
