🔍 Browser DevTools

Useful tools built into every browser — free, always there

🛠️ How to Open DevTools

Press F12 on Windows or Cmd + Option + I on Mac.
Or: right-click anywhere on a webpage → click Inspect.

A panel opens on the side or bottom. Don't be scared — you can't break anything!

📄 Elements Tab — See the HTML

The Elements tab shows all the HTML on the page as a live tree. Click any element to highlight it on the page.

You can even double-click text in the panel and edit it — changes show instantly (but reset when you refresh).

📋 View Page Source

Right-click anywhere on a webpage → click View Page Source. It opens the raw HTML file the browser downloaded.

Great for seeing how other websites are built — you can learn a lot by reading other people's code!

💬 Console Tab — Your JS Playground

The Console tab shows errors and lets you run JavaScript live. Add console.log() to your code to print values:

// in your JavaScript:
console.log("hello!");
console.log(score);
console.log("score is", score);

Those values appear in the Console tab. It's the easiest way to check what your code is doing!

🔴 Red Errors in the Console

If something isn't working, open the Console tab. Red text = an error. It shows you which file and which line number has the problem.

Copy the error message and search it online — someone has had the same problem before!
